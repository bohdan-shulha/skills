# qt-qml-attachments

Guidance for Qt/QML chat attachment rows and message bubble layout quirks.

## When to use

- Attachments stack vertically instead of wrapping in rows.
- Right/left alignment is wrong or ordering flips unexpectedly.
- Message bubbles stretch full width.

## What it covers

- Flow width and implicit sizing rules.
- Right-aligned wrapping with `layoutDirection`.
- Bubble width caps and attachment chip sizing.
